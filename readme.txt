#Summary
A Neighborhood map that displays an initial set of locations
with a flexible listview that changes depending on user input.

#Skills/Technologies used
1.html
2.css
3.javascript
4.jquery
5.Google Maps api
6.Foursquare api
7.Knockout.js
8.Ajax requests

#How to run on your machine
1. Download the zip the files
2. Extract all to whatever directory you want
3. Double click the index.html file and/or run it in your browser

#Future Features
1. Adding a new marker and list item using an autocomplete search function.
